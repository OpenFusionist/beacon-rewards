Field reference

  1. Window: Reporting window used (start to end)
  2. Active validator count: Total validators currently active
  3. CL rewards: Consensus layer rewards, including attestations and proposals (ACE)
  4. EL rewards: Execution layer rewards, primarily transaction fees (ACE)
  5. Total rewards: Sum of CL and EL rewards (ACE)
  6. Total effective balance: Effective balance across all active validators (ACE)
  7. Projected APR: Annualized rate based on the current window rewards (%)
